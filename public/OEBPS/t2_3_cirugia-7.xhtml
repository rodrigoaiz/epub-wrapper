Diagnóstico diferencial

Tratamiento

Para pacientes asintomáticos, mínimamente sintomáticos o que rechazan manejo quirúrgico, se considera vigilancia clínica periódica; aunque hasta el 70% terminan sometiéndose a cirugía en los siguientes cinco años. Para pacientes sintomáticos se recomienda cirugía electiva de reparación con malla, abordaje abierto o laparoscópico, este último se prefiere para hernias bilaterales (Simons et al., 2016).

En caso de presentar datos de estrangulación, se requiere manejo quirúrgico urgente, difiriendo el uso de mallas si se presenta un proceso séptico acompañante.
